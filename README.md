# Support Vector Machines (SVM) Kernel Analysis

## 📌 Overview
This project explores how different kernel functions (Linear, Polynomial, and RBF) affect the performance of Support Vector Machines (SVM).

## 📊 What this project includes
- Jupyter Notebook with full code
- Visualisations of decision boundaries
- Comparison of kernel performance

## ⚙️ How to run
1. Download the notebook
2. Install required libraries:
   - numpy
   - matplotlib
   - scikit-learn
3. Run all cells

## 📁 Files
- svm_kernel_analysis.ipynb → Code and experiments
- svm_kernel_analysis.pdf → Tutorial explanation

## 📚 Dataset
The Iris dataset is used (available in scikit-learn).
